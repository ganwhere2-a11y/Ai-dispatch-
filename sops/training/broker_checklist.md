# Broker Checklist — Call Cheat Sheet
**Source:** Owner's physical dispatch training document
**Use:** Every broker call. Start with this. Don't try to remember everything at once.

This saves you from forgetting key details, helps you sound more professional, and
keeps your dispatch process clean and repeatable.

> Don't try to ask everything at once — use this to guide the conversation.
> Start with the must-have info (✅), lock in the rate, then confirm the rest.

## IRON RULES CHECK (Before Confirming)

| Rule | Check |
|---|---|
| No Florida origin or destination | ✅ / ❌ |
| RPM ≥ $2.51 (reject if below) | ✅ / ❌ |
| Deadhead ≤ 50mi AND ≤ 25% loaded miles | ✅ / ❌ |
| Weight ≤ 48,000 lbs | ✅ / ❌ |
| Dry van only (no reefer/flatbed/hazmat) | ✅ / ❌ |

**Never let a truck move without a signed rate confirmation.**
